## âš¡ Getting Started
### Prerequisites
- Node.js  
- .NET 7 SDK  
- SQL Server  

### Installation
1. Clone the repository  
2. Navigate to the backend folder and run:
   dotnet restore
   dotnet ef database update
   dotnet run
Navigate to the frontend folder and run:
  npm install
  npm start
